# MicroPKI

Учебный проект по созданию минимальной, но полной инфраструктуры публичных ключей (PKI).

## Описание

MicroPKI - это CLI-инструмент для создания и управления собственной PKI, реализующий основные концепции: корневой CA, промежуточные CA, выпуск сертификатов, отзыв (CRL) и проверку статуса (OCSP).

## Структура проекта

````
micropki/
├── micropki/  
│ ├── init.py
│ ├── cli.py           # парсер аргументов
│ ├── ca.py            # логика корневого УЦ
│ ├── certificates.py  # Обработка X.509 сертификатов
│ ├── crypto_utils.py  # PEM, загрузка ключей, шифрование
│ └── logger.py        # Снастройка логирования
├── tests/             # файлы pytest      
│ ├── test_ca.py
│ ├── test_certificates.py
│ └── test_crypto_utils.py
├── requirements.txt    
└── README.md # Этот файл
````
